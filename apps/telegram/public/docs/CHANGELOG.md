# Полный список изменений (Changelog)
Период: v3.0.0 — v3.3.9

## Версия 3.3.x

### v3.3.9 - Управление аватарами
- **Feat**: Реализовано управление аватарами в настройках: добавление, удаление и просмотр.
- **Feat**: Интеграция с Google Analytics.
- **Fix**: Исправлен парсинг данных аватаров в API.
- **Refactor**: Обновлены переводы для подтверждения удаления.

### v3.3.6 - Партнерская программа
- **Feat**: Реализован сервис начисления бонусов партнерам.
- **Feat**: Интеграция бонусов в платежные шлюзы (Telegram Stars, Fiat).

### v3.3.5 - Реферальная система и Авторизация
- **Feat**: Внедрена реферальная система для регистрации.
- **Feat**: Новое модальное окно авторизации (Email, Google).
- **Feat**: Автоматическое создание пользователей при входе через OAuth.
- **Feat**: Привязка существующих аккаунтов по Email.

### v3.3.3 - Мультиязычность
- **Feat**: Добавлена поддержка языков: Русский, Английский, Немецкий, Французский, Испанский, Арабский.
- **Feat**: Компонент переключения языка в настройках.

### v3.3.2 - Монорепозиторий и Web
- **Refactor**: Переход на структуру монорепозитория (apps/telegram, apps/mobile).
- **Feat**: Вход через Google OAuth и Email.
- **Feat**: Ссылки на "Политику конфиденциальности" и "Условия использования".
- **Feat**: Улучшена поддержка PWA (Service Worker, установка).
